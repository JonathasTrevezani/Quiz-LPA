## 📖 Sobre o Projeto

Este é um **quiz interativo** desenvolvido para auxiliar no estudo da disciplina **Lógica de Programação e Algoritmos**. O projeto foi criado com base no material de estudo da Uninter (aulas 1 a 5), oferecendo uma ferramenta prática para autoavaliação e revisão dos conceitos fundamentais da programação.

### 🎯 Finalidade

- **Uso Acadêmico**: Complemento aos estudos da disciplina, permitindo que alunos testem seus conhecimentos de Lógica de Programação e Algoritmos de forma interativa
- **Uso Pessoal**: Ferramenta de autoavaliação para revisão de conteúdos antes de provas e avaliações
- **Recurso Didático**: Material de apoio para professores e tutores da disciplina

## ✨ Funcionalidades

- ✅ **Questões objetivas** baseadas nos conteúdos de Lógica de Programação e Algoritmos
- ✅ **Questões discursivas** para aprofundamento teórico
- ✅ **Navegação intuitiva** por números
- ✅ **Feedback imediato** com correção detalhada (resolução passo a passo)
- ✅ **Barra de progresso** para acompanhamento
- ✅ **Sistema de pontuação** (6,7 pontos por questão)
- ✅ **Design responsivo** (funciona em celulares, tablets e computadores)
- ✅ **Interface moderna** com gradientes e animações
- ✅ **Botão de reiniciar** para refazer o quiz
- ✅ **Seção de comentários discursivos** com gabarito para estudo

## 📚 Conteúdo Abordado

O quiz cobre os seguintes tópicos da disciplina:

| Ordem | Tema Principal | Exemplo de Questão |
|-------|----------------|---------------------|
| 1 | Conceitos de Algoritmos e Lógica | Definição de lógica e algoritmo |
| 2 | Representações de Algoritmos | Descrição narrativa, pseudocódigo e fluxograma |
| 3 | Arquitetura de Von Neumann | Partes principais do computador |
| 4 | Linguagem de Programação | Conceito de linguagem de máquina e compilação |
| 5 | História da Linguagem Python | Linguagem inspiradora e origem do nome |
| 6 | Características do Python | Bibliotecas, comunidade e usos |
| 7 | Tipos Primitivos de Dados | int, float, string, bool |
| 8 | Nomes de Variáveis | Regras para nomes válidos em Python |
| 9 | Operadores Lógicos | Comparações e operadores relacionais |
| 10 | Estruturas Condicionais | if, elif, else |
| 11 | Laços de Repetição (while) | Sintaxe e funcionamento |
| 12 | Laços de Repetição (for) | Sintaxe e funcionamento com range |
| 13 | Estruturas Aninhadas | Laços dentro de laços |
| 14 | Funções em Python | Definição, parâmetros e retorno |
| 15 | Escopo de Variáveis | Global x Local |

### 🧠 Temas Discursivos

Além das questões objetivas, o quiz conta com uma seção de **comentários discursivos** sobre os seguintes temas:

1. **Representações de Algoritmos** (Descrição Narrativa, Pseudocódigo e Fluxograma)
2. **Variáveis e Tipos Primitivos de Dados** (int, float, string, bool)
3. **Laços de Repetição** (while e for) e Estruturas de Controle
4. **Funções:** Definição, Parâmetros, Retorno e Escopo de Variáveis
5. **Estruturas Condicionais e Operadores Lógicos** (if, elif, else)
