1. Representações de Algoritmos (Descrição Narrativa, Pseudocódigo e Fluxograma):
Algoritmos podem ser representados de três formas principais fora de uma linguagem de programação:
• Descrição Narrativa: é a forma mais livre e natural, utilizando a língua portuguesa para descrever os passos. Por não ter formalismo, está sujeita a ambiguidades e interpretações duplas. Exemplo: "Primeiro, pegue o valor de entrada, depois multiplique por 2 e mostre o resultado."
• Pseudocódigo: é uma representação mais estruturada, com regras e formalismo definidos, que se aproxima de uma linguagem de programação, mas ainda utiliza uma linguagem natural (português ou inglês) para facilitar o entendimento. É amplamente adotado em livros didáticos e provas. Exemplo: "LEIA valor; resultado ← valor * 2; ESCREVA resultado".
• Fluxograma: é uma representação gráfica que utiliza símbolos padronizados (ISO) para representar as operações, decisões e fluxos de um algoritmo. É muito útil para visualizar a lógica de forma clara e intuitiva. Exemplo: losango para decisão, retângulo para processamento, paralelogramo para entrada/saída.

2. Variáveis e Tipos Primitivos de Dados (int, float, string, bool):
Variáveis são espaços na memória do computador que armazenam dados. Em Python, os tipos primitivos principais são:
• int (inteiro): armazena números sem casas decimais. Exemplo: idade = 25, ano = 2024.
• float (ponto flutuante): armazena números com casas decimais. Exemplo: salario = 1500.75, nota = 8.5.
• string (cadeia de caracteres): armazena textos, caracteres e símbolos. Exemplo: nome = "João", cidade = 'São Paulo'.
• bool (booleano): armazena valores lógicos True (verdadeiro) ou False (falso). Exemplo: aprovado = True, ativo = False.
A escolha correta do tipo de dado é essencial para garantir que o programa funcione adequadamente, evitando erros de conversão e perda de precisão.

3. Laços de Repetição (while e for) e Estruturas de Controle:
Laços de repetição são estruturas que permitem executar um bloco de código várias vezes.
• while (enquanto): repete um bloco enquanto uma condição for verdadeira. É usado quando não sabemos exatamente quantas iterações serão necessárias. Exemplo: while (x <= 10): print(x); x += 1.
• for (para): repete um bloco por um número determinado de vezes, geralmente usando um iterador. É usado quando sabemos quantas iterações precisamos. Exemplo: for i in range(1, 11): print(i).
• Estruturas de controle como break (interrompe o laço imediatamente) e continue (pula para a próxima iteração) podem ser combinadas dentro dos laços para controlar o fluxo de execução.
• Cuidados importantes: evitar loops infinitos (quando a condição do while nunca se torna falsa) e garantir que a variável de controle seja corretamente iterada.

4. Funções: Definição, Parâmetros, Retorno e Escopo de Variáveis:
Funções são blocos de código reutilizáveis que executam tarefas específicas.
• Definição: em Python, usa-se a palavra-chave def. Exemplo: def soma(a, b): return a + b.
• Parâmetros: são dados que a função recebe para executar sua tarefa. Podem ser obrigatórios ou opcionais (com valores padrão). Exemplo: def cadastro(nome, idade, salario=1000).
• Retorno: a função pode retornar um valor usando return. Funções sem return são chamadas de procedures e retornam None implicitamente.
• Escopo: variáveis locais existem apenas dentro da função onde foram criadas; variáveis globais existem em todo o programa. O escopo determina onde a variável pode ser acessada.
• Vantagens: reutilização de código, abstração (esconder complexidade), organização e manutenção facilitada.

5. Estruturas Condicionais e Operadores Lógicos (if, elif, else):
Estruturas condicionais permitem que o programa tome decisões com base em condições lógicas.
• if (se): executa um bloco se a condição for verdadeira. Exemplo: if (nota >= 7): print("Aprovado").
• elif (senão se): verifica uma nova condição se a anterior for falsa. Exemplo: elif (nota >= 5): print("Recuperação").
• else (senão): executa um bloco se todas as condições anteriores forem falsas. Exemplo: else: print("Reprovado").
• Operadores lógicos: and (e), or (ou), not (não) são usados para combinar condições. Exemplo: if (idade >= 18 and idade <= 65): print("Adulto").
• Cuidados: atenção à indentação (espaçamento) em Python, que define os blocos de código; e à ordem correta das condições, pois o elif é avaliado em sequência.
